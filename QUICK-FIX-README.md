# Quick Fix - Infinite Recursion Error

## ⚠️ Belangrijk: Gebruik ALLEEN het .sql bestand!

Je hebt waarschijnlijk per ongeluk een `.md` (markdown) bestand geplakt. 

## ✅ Juiste Stappen:

1. **Open het bestand**: `FIX-USERS-RECURSION.sql` (NIET het .md bestand!)

2. **Kopieer ALLEEN de SQL code** - vanaf de eerste regel tot de laatste regel

3. **Plak in Supabase SQL Editor** en klik "Run"

## 📋 Welk Bestand Moet Je Gebruiken?

- ✅ **Gebruik**: `FIX-USERS-RECURSION.sql`
- ❌ **Niet gebruiken**: `FIX-INFINITE-RECURSION-ACTIE.md` (dit is markdown, geen SQL!)
- ❌ **Niet gebruiken**: `FIX-INSTRUCTIES.md` (dit is markdown, geen SQL!)

## 🔍 Hoe Herken Je Het Juiste Bestand?

- **SQL bestanden** eindigen op `.sql`
- **Markdown bestanden** eindigen op `.md`
- SQL begint met `--` (comments) of `DROP POLICY` of `CREATE POLICY`
- Markdown begint met `#` (headings)
